# 神人游戏

一个猎奇烂梗自嗨小游戏

## 环境要求

- Python 3.6 或更高版本
- Pygame 2.5.2

## 安装步骤

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行游戏：
```bash
python main.py
```

## 游戏控制

- Esc暂停
- 方向键移动
- 空格攻击
## 开发日志

### 圆头耄耋（5.16新增）
传说中存在于古战场的哈基米，出场时会伴随耄耋A梦的BGM，已知具备以下特性：

1.由于猫的反应是人的七倍，所以在看到人时会应激使用七倍速冲刺狠狠地爱你

2.血量不健康时会飞升喵星，无视碰撞体积

3.飞升状态下会使用哈气攻击，造成远程伤害，弹体随飞行距离增大

4.击败后掉落耄耋的财产
